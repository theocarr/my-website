My Projects


Netball Data Visualization

A web app created with Javascript and D3 which visualizes the win and score data of teams competing in the ANZ Championship.

The Dark Within

A 3D horror game with random events and procedural generation. Developed in 2 months with a team consisting of both design and computer science students, using Unity and C#.

Bomber Bird

A simple 2D game published for Windows and Windows Phone 8. Made in one week for MSA using C# and Unity.

Mansion bandit

A multiplayer 2D first person adventure game featuring random room and item generation. Made in Java by a team of four.

Hot-seat Cluedo

A hot-seat multiplayer cluedo game made in Java.

Random Tweet Game

A tiny web app/game made for fun which uses the twitter api to get random tweets and gets you to guess who posted them.
